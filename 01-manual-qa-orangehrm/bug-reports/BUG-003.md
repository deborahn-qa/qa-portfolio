# BUG-003 — Leave Application Allowed Beyond Available Balance

| Field | Details |
|---|---|
| **Bug ID** | BUG-003 |
| **Title** | System allows leave application exceeding available balance without warning |

| **Priority** | High |
| **Status** | Open |
| **Related TC** | TC-LVE-008 |

---

## Description

An employee can submit a leave request for more days than their available leave balance without receiving any warning or validation error. The request is accepted and set to "Pending Approval," which could lead to HR processing requests that should be automatically flagged.

---

## Steps to Reproduce

1. Log in as an employee with a known leave balance (e.g., 5 days Annual Leave remaining)
2. Go to Leave > Apply
3. Select **Annual Leave** as leave type
4. Select a date range exceeding the available balance (e.g., 10 days)
5. Click **Apply**

---

## Expected Result

System displays a warning: _"Requested days exceed your available leave balance. Do you want to proceed?"_ — or blocks the submission entirely.

---

## Actual Result

Leave request is submitted successfully with status "Pending Approval." No balance warning is shown.

---

## Environment

| Item | Details |
|---|---|
| Browser | Chrome 120 |
| OS | Windows 10 |
| Reproducibility | 100% |

---

## Impact

HR administrators may unknowingly approve leave that exceeds entitlement, causing payroll and compliance issues.
