## Description

When a logged-in user clicks Logout via the user avatar menu, the application occasionally redirects back to the Dashboard instead of the Login page, maintaining an active session without re-authentication.

---

## Steps to Reproduce

1. Log in with valid credentials (Admin / admin123)
2. Navigate to any module (e.g., PIM > Employee List)
3. Click the user avatar icon in the top-right corner
4. Click **Logout**
5. Observe the redirect destination

---

## Expected Result

User is logged out and redirected to the Login page. Session is terminated.

---

## Actual Result

Intermittently, the user is redirected to the Dashboard with session still active. No login prompt is shown.
