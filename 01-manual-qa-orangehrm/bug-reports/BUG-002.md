## Description

When attempting to upload a profile photo for an employee, the system accepts a `.pdf` file without displaying an error or validation message. Only image formats (JPG, PNG, GIF) should be accepted.

---

## Steps to Reproduce

1. Log in as Admin
2. Navigate to PIM > Employee List
3. Open any employee profile
4. Click on the profile photo placeholder
5. Select a `.pdf` file from your system
6. Observe system behavior

---

## Expected Result

System displays a validation error: _"Invalid file type. Please upload JPG, PNG, or GIF."_

---

## Actual Result

File is accepted without error. No image is displayed (broken image placeholder), but no error message is shown to the user.
